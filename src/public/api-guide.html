🔄 Reconciliation
POST /api/reconcile
Trigger a new reconciliation run
ParameterTypeRequiredDescription
timestampToleranceSecondsnumberNoTolerance in seconds for timestamp matching (default: 300)
quantityTolerancePctnumberNoTolerance percentage for quantity matching (default: 0.01)
Example Request Body: 
{
  "timestampToleranceSeconds": 300,
  "quantityTolerancePct": 0.01
}
GET /api/reconcile/runs
Get all reconciliation runs with pagination
ParameterTypeRequiredDescription
limitnumberNoNumber of runs to return (default: 50)
offsetnumberNoNumber of runs to skip (default: 0)
